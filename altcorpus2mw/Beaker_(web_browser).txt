Beaker (web browser) &&&&& Introduction &&&&& &&&&& &&&&& '''Beaker''' is a free and open-source peer-to-peer web browser currently in development by Blue Link Labs. Beaker provides the user with tools for publishing and hosting websites directly from the browser; all files and websites are transferred using Dat, a hypermedia peer-to-peer protocol. The browser also supports the HTTP protocol. &&&&& &&&&& Beaker is built using the Electron framework and therefore uses the Chromium browser as a renderer for webpages. &&&&& &&&&& See also &&&&& * Dat (software) &&&&& * Comparison of web browsers &&&&& * Peer-to-peer web hosting &&&&& * Distributed computing &&&&& &&&&& References &&&&& &&&&& &&&&& &&&&& &&&&& &&&&& &&&&& &&&&& &&&&& &&&&&
